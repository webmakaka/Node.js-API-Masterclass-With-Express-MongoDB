## How to Run

<br/>

### Update Config

**api/config/config.env**

Need update MONGO_URI at least

<br/>

### Install packages

    $ npm install

<br/>

### Database Seeder For Bootcamps

    // ImportData
    $ node seeder -i

    // DestroyData
    $ node seeder -d

<br/>

### Run Application in development mode

    $ npm run dev
